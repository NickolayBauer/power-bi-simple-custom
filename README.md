## Nodejs должен быть установлен

- [Node js](https://nodejs.org/)

## Установка пакетов

```
npm install --save
```

## Запуск pbiviz

```
pbiviz start
```

## Билд компонента (будет собрано в ./dist/)

```
pbiviz package
```
